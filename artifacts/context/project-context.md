# Project Context

## Overview
**Spotify-Decision** is a Decision Transparency Portal for Spotify, built as a final deliverable for MIS 430 (Spring 2026). The client is Spotify.

## Deliverables
The final submission is a single webpage with three tabs:
1. **Demo Tab**: A working prototype of the Decision Transparency Portal, styled as if it were an actual Spotify feature
2. **Presentation Tab**: Slide deck rendered via reveal.js (content provided by group members)
3. **Appendix Tab**: Supporting explanations of how semester work informed the final website (content provided by group members)

## Key Constraints
- UI must be indistinguishable from Spotify's actual interface -- not a mimic, but as if it were a real Spotify feature
- No full backend required; mock data is acceptable
- Hosting target: DigitalOcean droplet
- Group members are providing presentation content and supporting info; our job is the webapp skeleton and demo
